# Feature Ideas

Captured ideas for future development. Not prioritized for current sprint.

## Parked Ideas

### Letter Editing After Generation
Let users edit AI-generated cover letters before copying/downloading. Could include a rich text editor or simple textarea with formatting.

### AI-Powered Semantic Job Matching (Gemini)
Send job descriptions + user skills to Gemini for semantic matching instead of keyword-based. More accurate but adds API cost and latency. Could be layered on top of the current client-side keyword matching as a "deep match" option.

### Job Alerts / Notifications
Let users set up saved searches and get notified (email or in-app) when new matching jobs appear on Arbetsformedlingen.

### PDF Export of Cover Letters
Generate downloadable PDF versions of cover letters with proper formatting, ready to attach to applications.

### Application Tracking
Track which jobs the user has applied to, with status (applied, interview, rejected, offer). Turn JobbJägaren into a full job hunt management tool.

### Multiple CV Support
Allow users to upload different CVs for different job types (e.g., one for admin roles, one for tech roles) and generate letters based on the relevant CV.

### Salary Insights
Pull salary data for matched occupations to help users understand market rates.

### Company Research
Auto-fetch company info (size, industry, reviews) for employers in job listings.
